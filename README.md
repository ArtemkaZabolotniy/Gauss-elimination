# Gauss-elimination

A simple implementation of Gaussian elimination for solving systems of linear equations.

## Overview

This repository contains code that demonstrates the Gaussian elimination method, a standard algorithm for transforming a system of linear equations into an upper-triangular form and then solving it by back-substitution.

## Features

- Solves systems of linear equations using Gaussian elimination
- Educational and easy to understand
- Lightweight repository structure

## Getting started

1. Clone the repository.
2. Open the project in your preferred development environment.
3. Run the program and provide the matrix / coefficients as input, if applicable.

## Algorithm summary

Gaussian elimination works in two main phases:

1. **Forward elimination** – convert the matrix into upper-triangular form.
2. **Back substitution** – compute the solution values from the last equation upward.
